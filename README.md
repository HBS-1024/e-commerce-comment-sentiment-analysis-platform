# 电商用户评论情感倾向分析系统

## 项目简介

这是一个基于Vue 3开发的电商用户评论情感分析系统，能够自动分析用户评论的情感倾向，帮助商家了解消费者对产品的真实反馈。

## 技术栈

- **前端框架**: Vue 3 + Composition API
- **构建工具**: Vite
- **图表库**: Chart.js
- **图标库**: Lucide Vue Next
- **样式**: CSS3 + Flexbox/Grid

## 核心功能

### 1. 情感分析
- 基于情感词典的中文情感分析算法
- 支持正面、负面、中性三种情感分类
- 计算情感分数和情感强度
- 识别情感关键词

### 2. 数据可视化
- 情感分布饼图
- 情感统计柱状图
- 实时更新图表数据

### 3. 评论管理
- 评论列表展示
- 商品筛选
- 情感类型筛选
- 关键词搜索
- 添加新评论
- 查看评论详情

### 4. 统计概览
- 正面评论数量及占比
- 负面评论数量及占比
- 中性评论数量及占比
- 总评论数统计

## 情感分析算法

### 算法原理

系统采用基于情感词典的文本分析方法：

1. **情感词典匹配**
   - 正面词库：包含50+正面情感词
   - 负面词库：包含50+负面情感词
   - 程度副词：用于修饰情感强度
   - 否定词：用于反转情感倾向

2. **情感分数计算**
   - 匹配正面词加分，匹配负面词减分
   - 根据程度副词调整分数权重
   - 检测否定词反转情感

3. **情感分类**
   - 分数 > 0.3：正面
   - 分数 < -0.3：负面
   - 其他：中性

## 功能使用说明

### 查看评论
1. 打开系统后自动加载所有评论
2. 可通过商品筛选、情感筛选、关键词搜索过滤评论
3. 点击评论卡片查看详细信息

### 添加评论
1. 点击右上角"添加评论"按钮
2. 填写商品名称、用户名、评分和评论内容
3. 点击提交，系统自动分析情感并添加到列表

### 查看分析结果
1. 查看顶部统计卡片了解整体情感分布
2. 查看图表区域的可视化数据
3. 点击评论详情查看具体的情感分析结果和关键词

## 系统特点

- **实时分析**: 添加评论后立即进行情感分析
- **可视化展示**: 图表直观展示情感分布
- **交互友好**: 响应式设计，支持移动端
- **算法透明**: 显示情感关键词和分析结果
- **易于扩展**: 模块化设计，便于添加新功能
